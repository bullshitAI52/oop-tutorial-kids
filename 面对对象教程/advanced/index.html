🛡️ 第一课：封装 (Encapsulation)

想象你的对象是一个**藏宝箱**，里面装满了金币。如果谁都能直接把手伸进箱子拿钱，那就太危险了！

封装 的魔法就是给宝箱加上一层“保护罩”。想要存取金币？对不起，不能直接拿，必须通过安全守卫（也就是专门的方法）来进行。

💡 秘语：在 Python 中，如果给属性名字前面加上两个下划线 __ ，就像给它披上了隐形斗篷。外面的人就无法直接修改它啦！
📜 魔法咒语怎么写？
宝箱.py 
class TreasureChest:
    def __init__(self):
        # 加了双下划线，金币变成了私有（隐形）的！
        self.__coins = 100 
        
    def add_coins(self, amount):
        if amount > 0:
            self.__coins += amount
            print("放入金币成功！")
        
chest = TreasureChest()
# 如果有人想直接抢：
chest.__coins = 0  # ❌ 报错！小偷找不到金币！

🐉 第二课：多态 (Polymorphism)

多态 听起来很复杂，其实就像一种魔法指令！

当你对不同的魔法生物喊出相同的咒语要求他们：“发声！”，**龙**会喷火咆哮，**猫头鹰**会咕咕叫，**魔法青蛙**会呱呱叫。

同一个指令，不同类型的对象会有不同的表现形式，这就是“多重形态”（多态）！

💡 秘语：只要这些动物都有 speak() 方法，我们就可以不用管它具体是什么动物，统统对它调用 speak()！
📜 多态魔法咒语
魔法时刻.py 
class Dragon:
    def speak(self): print("吼嗷哦嗷！🔥")

class Owl:
    def speak(self): print("咕咕！🌙")

# 同一个法术函数，接受任何一只生物！
def cast_voice_spell(creature):
    creature.speak()

dragon = Dragon()
owl = Owl()

cast_voice_spell(dragon) # 调用了龙的发声
cast_voice_spell(owl)    # 调用了猫头鹰的发声
