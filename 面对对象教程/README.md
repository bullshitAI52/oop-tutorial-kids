面对对象（面向对象）入门教程
- 目标：用简单的 Python 文件演示类、继承、封装、多态等基本概念。
- 结构：5 个独立脚本，每个脚本自成体系，最后用 demo.py 做一个小综合示例。
- 运行方式：进入 面对对象教程/ 目录，逐个执行 python <脚本名>.py。

文件列表
- basics.py        -> 简单的类与实例化示例
- inheritance.py   -> 继承与多态的基本演示
- encapsulation.py  -> 封装（私有属性、@property）示例
- polymorphism.py   -> 多态：同一接口，不同实现
- demo.py           -> 汇总演示，组合前面的示例

引导语
- 学习要点：理解 self、构造方法 __init__、方法、继承、属性封装、以及多态的概念。
- 每个脚本都可以独立运行，逐步加深理解。
