第 1 章什么是"对象"？🍎

你身边的每样东西都是一个对象：一只狗、一辆车、一个苹果……

每个对象有两样东西：

🟡 属性——描述它长什么样（颜色、名字、年龄……）

🟢 方法——描述它能做什么（叫、跑、吃……）

🌟 生活比喻：
一只狗 → 属性：名字叫"旺财"，颜色是黄色
            方法：会叫 🐶 → "汪汪！"

第 2 章什么是"类"？📋

类就是一张模板，用来批量制造对象。

比如："狗"这个类就是模板，用它可以造出很多只不同的狗。

🌟 生活比喻：
类 = 饼干模具 🍪
对象 = 用模具做出来的一块块饼干
每块饼干形状一样，但口味可以不同！
📝 用 Python 写一个"狗"类
🐍 dog.py
# 定义一个"狗"类（模板）
class Dog:

    # 初始化：给狗起名字
    def __init__(self, name, color):
        self.name  = name   # 属性：名字
        self.color = color  # 属性：颜色

    # 方法：让狗叫
    def bark(self):
        print(f"{self.name} 说：汪汪！")


# 用类造出两只狗（两个对象）
dog1 = Dog("旺财", "黄色")
dog2 = Dog("小黑", "黑色")

dog1.bark()   # 旺财 说：汪汪！
dog2.bark()   # 小黑 说：汪汪！

第 4 章继承：儿子像爸爸 👨‍👦

继承的意思是：小类可以"继承"大类的所有东西，还能加新功能！

比如：动物 是大类，狗 和 猫 都继承自动物。

🌟 生活比喻：
动物类 = 爸爸（会呼吸、会吃东西）
狗类 = 儿子（继承爸爸的能力，还会"汪汪叫"）
猫类 = 女儿（继承爸爸的能力，还会"喵喵叫"）
📝 继承的代码
🐍 animals.py
# 大类：动物
class Animal:
    def __init__(self, name):
        self.name = name

    def eat(self):
        print(f"{self.name} 在吃东西 😋")


# 狗 继承自 动物
class Dog(Animal):
    def bark(self):
        print(f"{self.name} 说：汪汪！🐶")


# 猫 继承自 动物
class Cat(Animal):
    def meow(self):
        print(f"{self.name} 说：喵喵！🐱")


d = Dog("旺财")
d.eat()    # 继承自 Animal
d.bark()   # Dog 自己的方法

c = Cat("咪咪")
c.eat()    # 继承自 Animal
c.meow()  # Cat 自己的方法
